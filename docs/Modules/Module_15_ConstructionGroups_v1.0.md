# Module_15_ConstructionGroups_v1.0

# Строительные группы

Дата фиксации: 19.06.2026

Статус: целевая версия модуля сформирована.

## Главная цель

Модуль 15 создает дополнительную классификацию материалов по этапам и частям
частного строительства.

Это не замена категориям материалов.

Категория отвечает на вопрос:

```text
Что это за товар?
```

Строительная группа отвечает на вопрос:

```text
Где этот материал применяется в строительстве?
```

## Принцип

Один материал может входить в несколько строительных групп.

Пример:

- пиломатериал может использоваться в опалубке, кровле и черновых работах;
- крепеж может относиться к фундаменту, кровле, фасадам и отделке;
- гидроизоляция может относиться к фундаменту, кровле и санузлам.

## Базовые группы

Начальный список:

- участок и подготовка;
- фундамент;
- коробка дома;
- перекрытия;
- кровля;
- фасады;
- окна и двери;
- инженерные сети;
- внутренняя черновая отделка;
- внутренняя чистовая отделка;
- бани и сауны;
- благоустройство;
- инструмент и расходники.

## MVP

В MVP нужно реализовать:

- справочник строительных групп;
- связь группы с категориями материалов;
- ручное назначение группы материалу;
- возможность одного материала в нескольких группах;
- фильтр материалов по строительной группе;
- использование групп в аналитике цен.

## Что не входит в MVP

Не делать сразу:

- автоматическую полную классификацию всех материалов;
- AI-принятие решений по группам без проверки;
- расчет норм расхода;
- формирование сметы только по группе без Модуля смет;
- сложные технологические зависимости.

## Ограничения

- Строительная группа не должна становиться товарной категорией.
- Товарная категория не должна становиться этапом строительства.
- Назначение группы должно быть проверяемым и редактируемым.
