# Module_02_KnowledgeBase_v1.0

## Цель

Хранить строительные знания, технологии и нормы, которые не принадлежат конкретному материалу.

## Бизнес-логика

- Нормы расхода зависят от технологии.
- Материалы связываются с нормами через технологический контекст.

## Сущности

- Technology
- ConsumptionNorm
- KnowledgeRule
- SourceReference

## Ограничения

- AI не создает нормы самостоятельно.
- Спорные нормы требуют подтверждения человеком.

## Будущие доработки

- Версионирование норм.
- Источники и доверенность правил.
