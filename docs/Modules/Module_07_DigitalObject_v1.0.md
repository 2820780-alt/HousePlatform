# Module_07_DigitalObject_v1.0

## Цель

Цифровой объект строительства и последующий цифровой двойник дома.

## Бизнес-логика

- Объект связывает пользователя, участок, проект, технологии, материалы, сметы, закупки, документы, фото и события.
- После строительства объект становится цифровым двойником дома.

## Сущности

- ConstructionObject
- ObjectEvent
- Document
- Photo
- Project

## Ограничения

- История объекта должна сохраняться.
- Документы и события должны быть связаны с объектом.

## Будущие доработки

- Эксплуатация дома.
- История обслуживания.
