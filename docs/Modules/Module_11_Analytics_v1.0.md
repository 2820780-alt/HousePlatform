# Module_11_Analytics_v1.0

## Цель

Аналитика цен, смет, закупок, поставщиков и строительства.

## Бизнес-логика

- Аналитика строится на накопленных данных платформы.
- Аналитические выводы должны быть объяснимыми.

## Сущности

- AnalyticsReport
- Metric
- PriceTrend
- Forecast

## Ограничения

- Аналитика появляется после данных и алгоритмов.
- AI может анализировать, но не является источником истины.

## Будущие доработки

- Прогноз бюджета.
- Сравнение регионов и поставщиков.
