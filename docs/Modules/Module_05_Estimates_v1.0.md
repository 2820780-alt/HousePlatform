# Module_05_Estimates_v1.0

## Цель

Создание, расчет и хранение смет.

## Бизнес-логика

- Смета собирает объемы, материалы, работы, доставку, отходы, запас и сроки.
- Расчет опирается на данные платформы, а не на неподтвержденный AI-вывод.

## Сущности

- Estimate
- EstimateItem
- Quantity
- EstimateVersion
- CalculationRule

## Ограничения

- Финансовые решения подтверждаются пользователем.
- Изменения сметы должны быть воспроизводимыми.

## Будущие доработки

- Версионирование смет.
- Сравнение вариантов.
