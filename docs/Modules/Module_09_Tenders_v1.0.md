# Module_09_Tenders_v1.0

## Цель

Организация тендеров между подрядчиками и поставщиками.

## Бизнес-логика

- Тендер создается на основе потребности объекта, сметы или закупки.
- Участники подают предложения, которые сравниваются по цене, срокам и условиям.

## Сущности

- Tender
- TenderLot
- TenderBid
- TenderParticipant

## Ограничения

- Выбор победителя подтверждает пользователь.
- История предложений сохраняется.

## Будущие доработки

- Рейтинги участников.
- Автоматическое сравнение условий.
