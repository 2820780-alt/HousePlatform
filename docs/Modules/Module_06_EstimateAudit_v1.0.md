# Module_06_EstimateAudit_v1.0

## Цель

Проверка смет на ошибки, дубли, пропуски и риски.

## Бизнес-логика

- Аудит сравнивает смету с нормами, ценами и историей изменений.
- AI может объяснять и предлагать варианты, но не является источником истины.

## Сущности

- EstimateAudit
- AuditFinding
- AuditRule
- RiskFlag

## Ограничения

- Спорные выводы требуют подтверждения.
- Аудит не должен автоматически менять смету.

## Будущие доработки

- Приоритизация рисков.
- Объяснения причин расхождений.
