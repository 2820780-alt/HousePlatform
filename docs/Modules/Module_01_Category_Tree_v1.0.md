# Module_01_Category_Tree_v1.0

Дата фиксации: 20.06.2026

Статус: рабочая иерархия для Module_01_Materials_v1.2.

Назначение документа: зафиксировать дерево категорий Material Hub, ключевые характеристики и правила классификации материалов для частного домостроения.

## Принципы

1. Категория отражает строительное назначение материала, а не название сайта-источника.
2. Подкатегория должна быть пригодна для фильтрации, сравнения цен и будущих смет.
3. Тип материала и характеристики используются для поиска дублей, аналогов, конвертации единиц и будущей базы знаний.
4. Общие категории вроде "Строительные материалы" не используются как рабочие категории.
5. Если источник отдает раздел каталога вместо карточки товара, материал не создается автоматически.
6. Если классификация спорная, материал попадает в "Требует классификации".

## Рабочее дерево категорий

### Стеновые материалы

Подкатегории:

- Газобетонные блоки.
- Лотковые блоки.
- Кирпич.
- Керамические блоки.
- Перемычки.
- Кладочные элементы.

Примеры:

- Газобетонный блок 625*200*250 D500.
- Лотковый блок 500*400*250 D500.
- Армированная перемычка.

Ключевые характеристики:

- длина;
- ширина;
- высота;
- плотность;
- класс прочности;
- назначение: стеновой, перегородочный, лотковый, перемычка;
- производитель;
- регион.

Правила классификации:

- "газобетон", "газобетонный блок", "стеновой блок", "Bonolit", "ВКБлок" -> Газобетонные блоки;
- "П-образный", "U-блок", "лотковый" -> Лотковые блоки;
- лотковые блоки не должны называться "Газобетонный блок", если из карточки ясно, что это блок для армопояса.

### Сухие смеси

Подкатегории:

- Клеи для кладки.
- Цементные шпатлевки.
- Гипсовые шпатлевки.
- Полимерные шпатлевки.
- Цементные штукатурки.
- Гипсовые штукатурки.
- Сухие строительные смеси.
- Грунтовки и добавки.

Примеры:

- Клей для кладки газобетона ВК-100.
- Шпатлевка цементная ВК-500.
- Штукатурка гипсовая 30 кг.

Ключевые характеристики:

- вес мешка;
- расход;
- основание;
- назначение;
- толщина слоя;
- жизнеспособность раствора;
- морозостойкость;
- производитель.

Правила классификации:

- "клей", "кладочная смесь", "ячеистого бетона" -> Клеи для кладки;
- "шпатлевка цементная" -> Цементные шпатлевки;
- "шпатлевка гипсовая" -> Гипсовые шпатлевки;
- "шпатлевка полимерная" -> Полимерные шпатлевки;
- "штукатурка цементная" -> Цементные штукатурки;
- "штукатурка гипсовая" -> Гипсовые штукатурки.

## Материалы, требующие отдельной проверки

В "Требует классификации" отправлять:

- карточки разделов каталога, а не товары;
- материалы без размеров, если размер является обязательным для дублей;
- KNAUF-профили без типа и размера;
- листовые материалы без толщины;
- крепеж без диаметра и длины;
- смеси без типа: шпатлевка, штукатурка, клей или другое;
- материалы, где название конфликтует с категорией источника;
- товары, для которых источник указал город или маркетинговый текст внутри названия.

## Связь с конвертером единиц

Для конвертации нужны подтвержденные характеристики:

- доска и брус: толщина, ширина, длина;
- газоблоки и лотковые блоки: длина, ширина, высота;
- OSB, ГКЛ, ГВЛ, фанера: длина, ширина;
- сухие смеси: вес мешка;
- кабель: длина бухты;
- крепеж: вес одной штуки.

Если характеристик нет, материал попадает в список "Требует данных для конвертации".

## Связь с будущей Knowledge Base

Material Hub хранит факты: материалы, характеристики, документы, ссылки, источники, цены и историю цен.

Данные о технологиях, расходе, совместимости, типовых узлах и шагах монтажа сохраняются только как KnowledgeCandidate и не становятся нормами автоматически.
