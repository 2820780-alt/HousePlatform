# Module_01_Materials_v1.0

## Цель

Нормализованная база строительных материалов.

## Бизнес-логика

- Материалы наполняются автоматически из прайсов, загрузок и внешних источников.
- Администратор модерирует спорные совпадения и новые карточки.
- Нормы расхода не хранятся в модуле материалов.

## Сущности

- Material
- MaterialCategory
- MaterialAttribute
- MaterialAlias
- Unit
- UnitAlias
- UnitConversion

## Ограничения

- Поиск дублей выполняется только внутри категории.
- AI не является источником истины и не объединяет спорные данные без подтверждения.

## Будущие доработки

- Правила классификации материалов.
- Качество карточек материалов.
- Механизм слияния дублей через модерацию.
