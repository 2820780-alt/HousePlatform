# HousePlatform

Платформа управления строительством частных домов.

Основная цель проекта — создание долгосрочной строительной data-платформы с накоплением данных по материалам, поставщикам, технологиям строительства, сметам, объектам строительства, эксплуатации домов и AI-аналитике.

## Статус проекта

Этап 0 — подготовка архитектуры.

## Основные направления

- база материалов;
- база поставщиков;
- база технологий;
- база нормативов;
- история цен;
- объекты строительства;
- сметы;
- закупки;
- AI-анализ;
- цифровой двойник дома.

## Принципы

- данные важнее интерфейса;
- все сущности должны быть связаны;
- история изменений должна сохраняться;
- проект должен быть переносимым;
- резервное копирование обязательно.
