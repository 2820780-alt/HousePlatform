# HousePlatform

Платформа управления строительством частных домов.

Текущая зафиксированная архитектурная версия: `Master_Prompt_v1.0`.

## Главные документы

- `docs/Master_Prompt_v1.0.md` — цели, принципы, архитектура, ограничения.
- `docs/Decisions.md` — архитектурные решения.
- `docs/Roadmap.md` — этапы разработки и приоритеты.
- `docs/Modules/` — версионированные описания модулей.

## Правило работы

`Master_Prompt_v1.0` не редактируется напрямую без фиксации новой версии.

Изменения в предметной области выполняются через:

- "Редактируем Модуль X"
- "Создаем версию 1.1 Модуля X"
- "Добавляем решение в Decisions"
- "Обновляем Roadmap"

## Технический контур

- Backend: FastAPI.
- База данных: PostgreSQL через SQLAlchemy async.
- Очереди/кеш: Redis, Celery.
- Временный UI-прототип и старый SQLite/SQLModel слой считаются устаревшими.

## MVP-фокус

В MVP попадает только то, что помогает пользователю получить смету, цены,
объемы, подрядчика, поставщика или контролировать стройку.
